## CONTROLES
<img src="../touch.jpeg" width="300"/>

**Switches**
- S07-S08 - Dron/teclado

**PADS**
- P01/P02 - UP/DOWN MODELOS
- P03-P10 - TECLADO

**Perillas**
- S11 - Frequency
- S34 - Harmonics
- S30 - Timbre
- S35 - Morph
- S37 - Escala
